# MapleSixCaculator
메이플스토리 6차 스킬을 효율적으로 찍는 순서를 알려주는 프로그램입니다.


1. 사용방법

먼저 우측 상단의 초록색 Code 버튼을 클릭 후 나오는 선택지 중 Download ZIP 버튼을 클릭해서 압축파일을 다운받고, 압축을 풀어줍니다. 
그리고 dist -> MapleSixSkilltreeCalculator 폴더 안에 있는 MapleSixSkilltreeCalculator.exe 파일을 실행하면 프로그램을 사용할 수 있습니다.
사전에 확인하셔야 할 내용은 스킬의 딜지분(전투분석을 이용해 분석), 마스터리코어와 스킬코어의 "0"레벨 데미지와 레벨당 데미지 상승률입니다(저는 나무위키에서 찾았어요).

1) 보총뎀과 방무를 입력하고 하단의 드래그박스를 이용해 효율계산을 원하는 재화를 선택합니다.
이 때 보총뎀과 방무는 오리진 스킬을 사용할 때의 보총뎀과 방무를 적어주시면 정확한 계산에 도움이 됩니다.
(에픽 어드벤처, 여신의 축복2, 직업별 버프 등 각종 버프스킬 , 도핑 , 파티버프 등 모두 포함)
(1.3 패치 추가) 1.1 조각의 효율성 계산(장기) , 2.1 솔 에르다의 효율성 계산(장기)가 드래그박스에 추가되었습니다.
간단하게 설명하면 오리진 30레벨을 빠르게 찍으실 수 있을 것 같으신 분은 1.1 , 2.1을 사용하시고 그렇지 않은 분들은 기존의 1 , 2를 사용하시면 됩니다.
자세한 설명은 폴더 안 patchnote.txt의 1.3패치에 4.2에 적어두었습니다.

2) 스킬 관련 정보를 각 칸에 입력합니다(이름 , 딜지분 , 데미지 관련 수치 , 레벨). 
이 때 딜지분은 xx.xx(13.52)와 같이 입력하고 데미지 관련 수치는 %를 제외하고 입력합니다. ex) 스킬 데미지 365 / 상승량 7
레벨은 입력하지 않을 시 5차 강화 코어는 0레벨(활성화하지 않음) , 나머지 코어는 1레벨(활성화만 하고 강화하지 않음)로 입력됩니다.

3) 계산 버튼을 클릭합니다.

4) 같은 폴더 안에 생성된 스킬트리(조각).txt 또는 스킬트리(솔 에르다).txt에서 결과를 확인합니다.


2. 스킬트리 읽는 방법

스킬 : 폭시 ｜ 레벨 : 1 -> 3
스킬 : 잔시 ｜ 레벨 : 0 -> 1
스킬 : 퀴버 ｜ 레벨 : 0 -> 1
스킬 : 폭시 ｜ 레벨 : 3 -> 5
스킬 : 레인 ｜ 레벨 : 0 -> 1
스킬 : 폭시 ｜ 레벨 : 5 -> 9
스킬 찍는 순서는
폭시 1 -> 3레벨, 잔시 0 -> 1레벨, 퀴버 0 -> 1레벨, 폭시 3 -> 5레벨, 레인 0 -> 1레벨, 폭시 5 -> 9레벨입니다.

3. 스킬트리 세부내용 활용방법

스킬 : 폭시   ｜ 레벨 : 1  -> 9  ｜ 최종뎀 증가율 : 3.1116129032 % ｜ 필요 조각 개수 : 192  ｜ 최종뎀 1%에 필요한 조각 개수 : 61.7043334024
스킬 : 오리진 ｜ 레벨 : 1  -> 4  ｜ 최종뎀 증가율 : 0.714989802  % ｜ 필요 조각 개수 : 105  ｜ 최종뎀 1%에 필요한 조각 개수 : 146.8552414321
스킬 : 잔시   ｜ 레벨 : 1  -> 2  ｜ 최종뎀 증가율 : 0.1254054054 % ｜ 필요 조각 개수 : 23   ｜ 최종뎀 1%에 필요한 조각 개수 : 183.40517241384

세부내용에 담긴 정보들을 활용하는 몇 가지 예시들을 소개해드리겠습니다. 

3_1) (최종뎀 1%에 필요한 조각 개수) * (조각의 현재 가격) = (최종뎀 1% 상승에 필요한 메소)

3_2) 해당 프로그램에서 계산되지 않는 특수한 옵션을 가진 5차 강화 코어의 강화 순서를 계산할 수 있습니다. 계산 순서는 다음과 같습니다.
3_2_1) 해당 5차 강화 코어의 특정 강화 구간에서의 최종데미지 상승률을 계산합니다.
ex) 5차 스킬 A의 강화코어를 3레벨에서 10레벨으로 올렸을 때 최종데미지 증가율 5%
3_2_2) 해당 5차 강화 코어의 강화에 필요한 조각 개수를 구합니다.
ex) 5차 강화 코어를 3레벨에서 10레벨로 강화시키는데 필요한 조각의 개수 388개
3_2_3) 3_2_2의 결과 / 3_2_1의 결과를 계산해 최종뎀 1%에 필요한 조각 개수를 구합니다.
ex) 388 / 5 = 77.6
3_2_4) 세부내용의 최종뎀 1%에 필요한 조각 개수를 확인하고 강화 순서를 변경해 코어를 강화한다.
ex) 77.6은 예시의 폭시 1 -> 9의 개수 61.7보다는 크고 , 오리진 1 -> 4의 개수 146.8보다는 작으므로
스킬 강화 순서를 폭시 1 -> 9 , A 3 -> 10 , 오리진 1 -> 4 , 잔시 1 -> 2로 변경해 코어를 강화한다.

이외에도 사용할 수 있는 방법은 개인의 활용법에 따라 다양한 방법들이 존재할 수 있습니다.


4. 주의사항

3_1) 본 프로그램은 5차 강화 코어를 제외한 나머지 코어(마스터리 코어, 오리진 등)를 활성화한 뒤 사용하셔야 합니다.

3_2) 5차 강화코어에서 스킬의 레벨을 올릴 때 최종뎀 111%, 112%, 113%.... 형태의 최종데미지 상승만 계산됩니다.
강화 코어에 붙은 지속시간 증가, 쿨타임 감소, 최종데미지 상승량 증가 등의 추가 옵션들은 계산되지 않습니다.
