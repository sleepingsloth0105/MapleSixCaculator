# MapleSixCaculator
메이플스토리 6차 스킬을 효율적으로 찍는 순서를 알려주는 프로그램입니다.

1. 사용방법

dist/Maple_User_interface 폴더 안에 있는 Maple_User_interface.exe 파일을 실행해서 사용합니다.
사전에 확인하셔야 할 내용은 스킬의 딜지분(전투분석을 이용해 분석), 마스터리코어와 스킬코어의 "0"레벨 데미지와 레벨당 데미지 상승률입니다(저는 나무위키에서 찾았어요).

1)보총뎀과 방무를 입력하고 하단의 드래그박스를 이용해 효율계산을 원하는 재화를 선택합니다.
이 때 보총뎀과 방무는 오리진 스킬을 사용할 때의 보총뎀과 방무를 적어주시면 정확한 계산에 도움이 됩니다.
(에픽 어드벤처, 여신의 축복2, 직업별 버프 등 각종 버프스킬 , 도핑 모두 포함)
(1.3 패치 추가) 1.1 조각의 효율성 계산(장기) , 2.1 솔 에르다의 효율성 계산(장기)가 드래그박스에 추가되었습니다.
간단하게 설명하면 오리진 30레벨을 빠르게 찍으실 수 있을 것 같으신 분은 1.1 , 2.1을 사용하시고 그렇지 않은 분들은 기존의 1 , 2를 사용하시면 됩니다.
자세한 설명은 폴더 안 patchnote.txt의 1.3패치에 3.2에 적어두었습니다.

2)스킬 관련 정보를 각 칸에 입력합니다(이름 , 딜지분 , 데미지 관련 수치 , 레벨). 
이 때 딜지분은 xx.xx(13.52)와 같이 입력하고 데미지 관련 수치는 %를 제외하고 입력합니다. ex) 스킬 데미지 365 / 상승량 7
레벨은 입력하지 않을 시 5차 강화 코어는 0레벨(활성화하지 않음) , 나머지 코어는 1레벨(활성화만 하고 강화하지 않음)로 입력됩니다.

3) 계산 버튼을 클릭합니다.

4) 같은 폴더 안에 생성된 스킬트리(조각).txt 또는 스킬트리(솔 에르다)에서 결과를 확인합니다.


2. result.txt 읽는 방법
결과가 다음과 같이 나왔다면
스킬 : 시작 / 레벨 : 0 -> 0
스킬 : 폭시 / 레벨 : 1 -> 3
스킬 : 잔시 / 레벨 : 0 -> 1
스킬 : 퀴버 / 레벨 : 0 -> 1
스킬 : 폭시 / 레벨 : 3 -> 5
스킬 : 레인 / 레벨 : 0 -> 1
스킬 : 폭시 / 레벨 : 5 -> 9
스킬 찍는 순서는
폭시 1 -> 3레벨, 잔시 0 -> 1레벨, 퀴버 0 -> 1레벨, 폭시 3 -> 5레벨, 레인 0 -> 1레벨, 폭시 5 -> 9레벨입니다.


3. 주의 

3.1 본 프로그램은 5차 강화 코어를 제외한 나머지 코어(마스터리 코어, 오리진 등)을 활성화한 뒤 사용하셔야 합니다.
3.2 현재 5차 강화코어에서 스킬의 레벨을 올릴 때 최종뎀 111%, 112%, 113%....의 형태로 스킬이 강화되지 않는 스킬의 효율성은 계산되지 않습니다.


4. 추가할 기능
해당 스킬을 강화할 때 최종뎀 1% 상승에 사용되는 조각, 솔 에르다의 개수와 해당 강화로 인해 상승하는 최종뎀을 같이 출력하는 기능을 추가할 예정입니다.
