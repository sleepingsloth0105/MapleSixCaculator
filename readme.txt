# MapleSixCaculator
메이플스토리 6차 스킬을 효율적으로 찍는 순서를 알려주는 프로그램이다.

1. 다운로드 및 사용방법

dist/Maple_User_interface 폴더 안에 있는 Maple_User_interface.exe 파일을 실행해서 사용한다.

1)보공과 방무를 입력하고 하단의 드래그박스를 이용해 버전을 선택한다.
2)스킬 관련 정보를 각 칸에 입력합니다. 이 때 딜지분은 xx.xx(13.52)와 같이 입력하고 데미지 관련 수치는 %를 제외하고 입력한다.
ex) 1레벨 데미지 365 / 상승률 7
3) 계산 버튼을 클릭한다.
4) 같은 폴더 안에 생성된 result.txt에서 결과를 확인한다.

2. result.txt 읽는 방법
결과가 다음과 같이 나왔다면
스킬 : 시작 / 레벨 : 0
스킬 : 폭시 / 레벨 : 9
스킬 : 오리진 / 레벨 : 4
스킬 : 잔시 / 레벨 : 2
스킬 : 퀴버 / 레벨 : 2
스킬 : 오리진 / 레벨 : 5
스킬 : 잔시 / 레벨 : 3
스킬 찍는 순서는
폭시 9레벨 -> 오리진 4레벨 -> 잔시 2레벨 -> 퀴버 2레벨 -> 오리진 5레벨 -> 잔시 3레벨이다.

3. 추가할 기능
현재는 솔 에르다 조각의 효율성만을 고려하는데
솔 에르다의 효율성을 계산하는 모드를 추가할 예정이다.

4. 주의 : 해당 프로그램은 모든 코어를 활성화시켰다는 전제로 작동합니다.
코어를 활성할 때의 효율은 계산되지 않습니다.
즉 모든 코어의 레벨은 1부터 시작합니다.
